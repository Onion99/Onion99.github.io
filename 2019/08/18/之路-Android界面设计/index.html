Toast

简单使用

Toast.makeText(context, text, duration).show;

  • context
    • getApplicationContext() - 返回所有活动的上下文,即返回应用的上下文
    • getBaseContext() - 返回由构造函数指定或setBaseContext()设置的上下文,一个context访问另一个context使用
    • getContext() - 返回当前活动的上下文
  • text
    • 展示信息
  • duration
    • Toast.LENGTH_SHORT - delay 2000
    • Toast.LENGTH_LONG - delay 3500
    • milisecond - (eg. 4000)

自定义控件

  • 自定义控件,所有控件都是直接或间接继承自View,所有布局都直接或间接继承ViewGroup
  • View是Android中最基本的一种UI组件,可以绘制图形,并响应事件
  • ViewGroup可以包含很多子View和子ViewGroup,用于放置控件与布局

布局属性

sp,dp,以及屏幕像素密度

  • px,即像素,一个像素单位对应一个屏幕像素单位,不会根据屏幕密度自动缩放
  • dp,即密度无关像素,自动缩放
  • sp,即缩放无光像素,受用户字体偏好设置的影响,通常用于设置屏幕字体

边距(margin)与内边距(padding)

  • 编剧决定组件的距离
  • 内边距决定组件距离所包含内容的大小

样式(style),主题(theme),主题属性

  • 样式,是对组件行为和外观的属性定义
  • 主题,即各种样式的集合,从结构上主题本身也是一种样式资源,只不过它的属性指向了其他样式资源
