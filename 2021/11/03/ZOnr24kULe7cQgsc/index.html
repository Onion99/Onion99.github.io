用来处理不常见的二维变换

  • Canvas.setMatrix(matrix)
    • 用 Matrix 直接替换 Canvas 当前的变换矩阵，即抛弃 Canvas 当前的变换

    • 用 Canvas 当前的变换矩阵和 Matrix 相乘，即基于 Canvas 当前的变换，叠加上 Matrix 中的变换
